Huang’s Termination Detection Algorithm

Example of algorithm execution

Notations

  • B(DW) = Computation message sent as a part of the computation and DW is the weight assigned to it.
  • C(DW) = Control message sent from the process to the controlling agent and DW is the weight assigned to it.
  • John is the Controlling Agent that can send Computation Messages, and receive Control Messages.
  • Zoe, Leo and Basil are processes that can receive Computation Messages, and send Control Messages to John (Controlling Agent)
